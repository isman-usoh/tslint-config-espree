> A [TSLint config](https://palantir.github.io/tslint/usage/configuration/) for Espree.IO

## Installation

```bash
yarn add tslint-config-espress -D
```

## Usage

In `tslint.json`:

```json
{
  "extends": "tslint-config-espress"
}
```

### Rules

* [tslint](https://www.npmjs.com/package/tslint)
